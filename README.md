# GifTastic

### About this Project

Search for GIFs using the Giphy API. User creates buttons that, when clicked, displays the results as images and show the GIF when hovered over.

  **Visit Web App [Here!](https://dojeda1.github.io/GifTastic/)**

### Technologies used

* HTML
* CSS
* JavaScript
* jQuery
* Bootstrap
* Giphy API
